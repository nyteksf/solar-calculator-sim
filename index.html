Instructions for This Calculator:

Project Summary:

This web app was made to represent a fully functioning replica of a basic solar-powered calculator, and was inspired by a freeCodeCamp challenge. Its usage supports the various basic mathematical functions. Feel free to additionally switch off the left-adjacent light switch to to see what happens, if curious.

Special Function Keys:

ON/AC: If currently 'off', then it initializes the calculator. If calculator is already 'on', clears all variables and the screen itself.

OFF: Switches the calculator to its 'off' state.

EC: Clears only the last inputted variable as a whole.

This Virtual Calculator Supports the Following Operations:

Basic Operations:

  • Addition ('+'): Performs addition
  • Subtraction ('-'): Performs subtraction
  • Multiplication ('*'): Performs multiplication
  • Division ('/'): Performs division
  • Pos/Neg ('+/-'): Performs sign change
  • Decimal ('.'): Performs addition of a decimal point
  • Equals ('='):
      Function:
    • If you're adding together two given integers, then upon the completion of their input you may press the 'equals' key to return your result to the screen in the normal, expected fashion.
    • If you wish to continue your equation in the form of a flowing chain of operations, press an operator button after pressing equals and then another number. Similarly, begin entering in a third or a fourth number, and so forth, ad infinitum.
    • Worth noting is a third option of pressing the equal button after chaining together any number of the above operations. In such a case the result of your growing equation will accumulate without any need for additional pressing of equals. This math is handled automagically by the code.
